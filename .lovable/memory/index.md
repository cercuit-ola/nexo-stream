# Project Memory

## Core
Dark fintech dashboard. Primary #00D98B (green), bg #0A0A0A, cards #131B24. Inter body, Space Mono for numbers.
React + Vite + Supabase. No Next.js. Stellar mainnet for scheduler escrow.
NexolPay - crypto dashboard with scheduler, vault, gift card, virtual card features. No wallet/off-ramp.
